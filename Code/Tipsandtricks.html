OneLogin Tips & Hotkeys

Hotkey Action Use for
Ctrl + Alt + Shift + B Open Box from taskbar Search client name or other files without having to open File Explorer or Browser Window

Tips

1. For Transmittal Letter, Individual Reconciliation, or Prep Notes & Missing Documents, open in browser and download the desired file so you don't run the risk of moving any files without knowing.

2. If you accidentally mess up file from Box, go to browser window and you can recover file from Box or go back to desired previous file version.
